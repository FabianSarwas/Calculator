📱 Taschenrechner (Web)

Ein einfacher, browserbasierter Taschenrechner, der mit HTML, CSS und JavaScript umgesetzt wurde.
Das Projekt demonstriert grundlegende Webentwicklung sowie einfache Interaktionslogik im Frontend.

🚀 Features

Grundrechenarten (Addition, Subtraktion, Multiplikation, Division)
Dynamische Anzeige der Eingabe
Ergebnisberechnung per Klick
Löschen der letzten Eingabe
Responsives Layout mit Tabellenstruktur

🛠️ Technologien

HTML5 – Struktur der Anwendung
CSS3 – Styling und Layout
JavaScript – Logik und Interaktivität

📂 Projektstruktur

Taschenrechner/
│── index.html     # Hauptseite mit Logik
│── style.css      # Styling der Oberfläche
